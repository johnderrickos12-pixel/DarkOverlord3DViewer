## Dark Overlord 3D Viewer (HTML/Three.js)

This project provides a single `index.html` file that uses the Three.js library (via CDN) to render a simplified, programmatic 3D model of the "Dark Overlord" character directly in your web browser. No complex build steps or local server required.

### How to view:

Simply open `index.html` in your web browser.
